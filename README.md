# 音乐推荐系统 python759

本项目是一个基于Python的音乐推荐系统，融入了可视化分析与音乐数据挖掘功能，适用于软件工程与大数据领域。

## 技术栈

- **后端**: Python
- **数据库**: MongoDB
- **前端**: HTML, CSS, JavaScript
- **数据分析**: Pandas, NumPy, Matplotlib, Seaborn
- **推荐算法**: 基于用户行为的协同过滤

## 功能模块

- **用户行为分析**
- **音乐数据挖掘**
- **推荐算法实现**
- **可视化展示**
- **系统管理**

## 系统角色

- **普通用户**: 可接受音乐推荐，进行音乐播放等操作。
- **管理员**: 负责系统维护、数据管理等任务。

## 运行环境

- Python 3.6+
- MongoDB 4.0+
- 浏览器兼容性: 谷歌Chrome最新版优先支持

## 部署步骤

1. 安装Python环境和必要的库
2. 安装并启动MongoDB服务
3. 部署后端服务
4. 部署前端页面
